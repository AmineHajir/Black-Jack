structure de données utilisé pour le jeu black jack 

tableau de 52 cases. chaque case représente une carte de jeu. chaque case du tableau peut avoir 4 valeurs possibles \-: 0,1,2 ou 3. la valeur d'une case vaut 0 si la carte correspondante n'a toujours pas été tirée la valeur d'une case vaut 1 si la carte correspondante a été tirée par la banque et que cette carte est visible par le joueur la valeur d'une case vaut 2 si la carte correspondante a été tirée par le joueur la valeur d'une case vaut 3 si la carte correspondante a été tirée par la banque et que cette carte n'est pas visible par le joueur
